# 群島生態系 SPEC (原始綠色主題版)

> 此為備份版本，程式碼位於 `original-design` 分支
> 主分支已重構為「島嶼主題」，本檔案僅供歷史參考

---

## 1. Concept & Vision

**長照服務媒合平台** — 綠色森林系設計風格，象徵長照服務如同森林般呵護每一位長者。清新、專業、值得信賴的視覺感受。

---

## 2. Design Language

### 色彩系統
| Token | Hex | 用途 |
|-------|-----|------|
| `forest` | `#166534` | 主色，深森林綠 |
| `forest-light` | `#22C55E` | 亮綠 |
| `leaf` | `#86EFAC` | 淺綠葉色 |
| `bark` | `#78350F` | 棕色，樹幹/強調 |
| `earth` | `#F5F0E8` | 大地米色背景 |
| `sky` | `#0EA5E9` | 藍天色作為輔助 |

### 字體
- **標題**: `Noto Sans TC`, 700 weight
- **內文**: `Noto Sans TC`, 400 weight

### 背景主題
- 主背景：大地米色 `#F5F0E8`
- Header：白色 + 綠色漸層 border
- 主色調：森林深綠 `#166534`

## 3. Pages

### 3.1 首頁 (index.html)
- 頂部 Nav：品牌名稱 + 導航連結
- Hero 區：背景漸層 `#166534 → #22C55E`
- 服務項目大分類卡片（4項）
- 精選照服員展示
- 島嶼補給站商品推薦
- Footer 深色背景

### 3.2 探索群島 (caregivers.html)
- 網格佈局（響應式卡片）
- 照服員卡片列表（頭像 + 名稱 + 專長 + 評分 + 價格）
- 服務分類篩選標籤
- 搜尋框
- 無地圖/群島視覺，純列表形式

### 3.3 島主專頁 (caregiver-profile.html)
- 大圖片區塊頂部
- 頭像 + 姓名 + 評分 + 服務區域
- 服務項目列表
- 收費標準
- 評價展示
- 預約按鈕

### 3.4 登入 (login.html)
- 白色卡片居中
- 雙 Tab：登入 / 註冊
- 表單：Email + 密碼
- 第三方登入按鈕

### 3.5 預約 (booking.html)
- 步驟式表單
- 月曆 + 時段選擇
- 個案資料填寫
- 確認/成功頁

### 3.6 儀表板 (dashboard.html)
- 白色背景
- 統計數據卡
- 待處理預約列表
- 快捷連結

### 3.7 設定 (profile.html)
- 分頁 Tab 設計
- 基本資料編輯
- 服務項目設定

### 3.8 商店 (shop.html)
- 網格商品卡
- 分類側邊欄或頂部篩選
- 加入購物車功能

### 3.9 購物車 (cart.html)
- 商品列表
- 數量調整
- 優惠碼
- 結帳總金額

### 3.10 商品頁 (product.html)
- 單欄產品圖 + 資訊
- 加入購物車

### 3.11 管理後台 (admin.html)
- 白色背景管理介面
- 側邊導航（綠色）
- 數據儀表板
- 表格列表

## 5. Technical Approach

- **Framework**: Vanilla HTML + Tailwind CSS (CDN)
- **Fonts**: Google Fonts (Noto Sans TC)
- **State**: localStorage
- **No backend**: 純前端展示

## 6. 與島嶼主題的差異

| 項目 | 原始主題 | 島嶼主題 |
|------|---------|---------|
| 背景 | 大地米色 `#F5F0E8` | 深海藍 `#0A1628 → #16425B` |
| 主色 | 森林深綠 `#166534` | 海洋青 `#0891B2` |
| 探索頁 | 卡片列表 | 地圖 + 23座可點擊小島 |
| 島嶼元素 | 無 | 沙灘、草地、樹林、陰影 |
| 裝飾 | 簡單陰影 | 浮動動畫、海浪、格線 |
| 群島體驗 | 無 | 點擊島嶼展開 Modal |
| 配色 | 綠色系 | 藍色海洋 + 珊瑚橙強調 |
