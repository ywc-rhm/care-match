# 長照服務媒合平台 - 智慧照護Match

## 1. 平台願景

一個連結照服員與服務個案的媒合平台，同時整合長照輔具選購功能。讓服務媒合更順暢、輔具取得更便利。

**核心價值：** 
- 照服員：輕鬆管理可供服務的時間與項目
- 民眾/家屬：快速找到信賴的照服員與合適的輔具
- 透明評價機制，建立長照服務的信賴感

---

## 2. 功能模組

### 2.1 會員系統
| 角色 | 功能 |
|------|------|
| 照服員 | 註冊登入、個人檔案、服務項目設定、可服務時段管理、預約訂單管理、評價查看 |
| 消費者 | 註冊登入、搜尋照服員、預約服務、訂單管理、購買輔具、評價服務 |
| 管理者 | 審核照服員資格、管理會員、訂單管理、輔具商品上下架 |

### 2.2 服務媒合（核心功能）
- **搜尋過濾**：地區、服務類型、價格、評價、等級
- **時段卡**：視覺化週曆，展示照服員可服務時段
- **快速預約**：選擇服務項目 + 日期時段 + 個案需求描述
- **預約確認**：待確認 → 已確認 → 已完成 的狀態管理

### 2.3 長照小輔具商城
- 商品分類：行動輔具、衛浴輔具、居家安全、康復器材
- 商品卡：圖片、價格、規格簡述、庫存狀態
- 購物車 + 結帳流程
- 訂單查詢

### 2.4 評價系統
- 完成服務後雙方互評（1-5星 + 文字評論）
- 照服員平均評價顯示

## 4. 頁面結構

```
首頁
├── 服務搜尋列
├── 快速服務分類
├── 精選照服員
├── 熱門輔具推薦

/cart              - 購物車
/checkout           - 結帳

/dashboard          -會員專區（照服員/消費者儀表板）
/history            - 歷史訂單/預約
/profile           - 個人檔案
```

## 5. 技術方案

### Phase 1: GitHub Pages 靜態托管
- **前端**：HTML5 + TailwindCSS + Vanilla JavaScript
- **資料**：LocalStorage（展示用）+ Firebase（可擴充）
- **地圖**：Leaflet.js（免費開源）
- **部署**：GitHub Pages（免費）

### Phase 2: 遷移至 NAS
- **容器化**：Docker Compose
- **後端**：Python Flask 或 Node.js Express
- **資料庫**：SQLite（初期）/ PostgreSQL（成長後）
- **認證**：JWT + Session
- **反向代理**：Nginx
